Hyperion Data Bundle Workbench

Create, inspect, and validate data-driven certification/testing bundles without changing kernel code.

Provisioning Steps

1. Edit bundle data.
2. Protect trust-anchor data.
3. Run krn_certification_bundle --validate.
4. Attach external lab/scheme/device evidence.

This workbench is static and local. It never uploads profile, CAPK, vector, or evidence data.
